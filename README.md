Данный проект демонстрирует этапы ручного тестирования на базе крупного проекта Хекслета - Codebattle. Сам проект интересен и представляет собой игровое пространство для программистов, с возможностью живого общения в чате, вариантами соревноваться на любом языке программирования с различными противниками, как с друзьями, так и с рандомным пользователем или встроенным ботом. Дополнительный стимул для игроков - наличие индивидуальной статистики, а также общего рейтинга лидерующих в соревнованиях пользователей. Пользователи также активно участвуют в развитии проекта, активно поддерживая связь с разработчиками через сообщество. Сайт проекта Codebattle находится по адресу https://codebattle.hexlet.io/. В репозитории проекта qa-engineer-project-85 расположены файлы: issues.yml, reopened_issues.yml, requirements.yml, test-cases.yml, testing-report.yml, README.md. Проект создается на базе навыков, полученных при обучении в онлайн-школе Hexlet. Наставник, консультирующий по вопросам обучения: Фадеева Александра. Автор проекта: Асонова Анна.
